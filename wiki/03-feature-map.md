# 功能地图

| 区域 | 主要任务 | 数据输出 |
| --- | --- | --- |
| 今天 | 回顾近况、快速记录、继续未完成作品 | JournalEntry |
| 私密记录室 | 编辑事件、图片、意义、地点、关系、能量、意象和公开级别 | JournalEntry + NarrativeSignals |
| 旅程 | 探索记录生成的情绪世界与记忆场景 | WorldProgress + WorldDelta |
| 故事工坊 | 选择记录、编排场景、编辑转场和旁白 | StoryProject |
| 展厅 | 浏览作品、意象收藏、相册墙和书架 | StoryProject 视图 |
| 成长 | 查看非评判性的记录趋势、意象和成就 | 派生统计 |
| 设置 | 主题、动效、音频、输入和数据管理 | ThemeProfile + GameSettings |

桌面与移动端功能保持一致；桌面使用多栏编辑与快捷键，移动端使用分步编辑、底部操作区和触摸手势。
